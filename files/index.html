01 · about

I'm a first-year Software Engineering student at the Federal University of Technology, Minna — and a backend developer still early in the stack, on purpose.

I spend most of my time on the parts of an app users never see directly: auth flows, database schemas, payment webhooks, admin tooling. I like that backend work has a right answer — the request either resolves the way you expect or it doesn't. Right now I'm building with Node.js, Express, and MongoDB, and learning how real products handle payments, sessions, and the messy edge cases that only show up once something actually ships.
